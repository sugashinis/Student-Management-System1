# Student Management System

This is a Java-based Student Management System developed using JDBC.
The project helps manage student and teacher information with basic operations.

## Features
- Student login
- Teacher login
- Add student
- Remove student
- Update student marks
- View student details

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## How to Run the Project
1. Download or clone this repository.
2. Open the project in Eclipse IDE.
3. Configure database connection details in JDBC.java.
4. Create required tables in MySQL.
5. Run the main Java classes.
